Go back to previous commits:
git checkout commit-hash    → Go to specific commit
git checkout 9a8ee75        → Go back to old commit
git checkout HEAD~1         → Go to previous commit

Switch branches:
git switch -                → Switch to last branch

Restore or discard file changes:
git checkout HEAD namefile → Restore file from last commit
git checkout -- namefile   → Discard file changes
git restore nameFile       → Discard file changes
git reattach --source HEAD~2 fileName
git restore --stage secrets.txt

    1. Stay in detached head to view files ext.
    2. Leave and go back to wherever you were berofe reattach the head
