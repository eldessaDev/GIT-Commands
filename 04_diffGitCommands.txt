git diff                    → Compare changes with file not staged
git diff HEAD               → Compare changes with file in staged
git diff HEAD <file>        → Show specific file
git diff --staged           → Shows staged changes
git diff --staged <file>    → Show staged cheanges on specific file 
git diff branch1..branch2   → Compare changes between 2 branches
gif diff commit1..commit2   → Compare changes commits
