======================
=🧰ESSENTIAL-COMMANDS=
======================

📊 Repository Info
git status                  → Show current repository state (staged, unstaged, untracked files).
git log                     → Display full commit history.
git log --oneline           → Show concise commit history (1 line per commit).
git init                    → Initialize a new Git repository.

✅ Staging and Committing
git add file1 file2         → Stage specific files.
git add .                   → Stage all modified and untracked files.
git commit -m "message"     → Commit staged files with a message.
git commit -am "message"    → Stage and commit tracked files in one step.
git commit --amend          → Modify the last commit (message or files).

🗑️ Delete Git Repository
rm -rf .git                 → Completely delete Git tracking (removes .git folder).
